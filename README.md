# 🍽️ 점심메뉴 추첨기 (네이버 API 버전)

아일렉스 근처 300m 이내 음식점을 네이버 지역 검색 API로 실시간 검색하고 랜덤 추첨하는 웹 애플리케이션입니다.

## 📋 필요한 것

- Python 3.7 이상
- 네이버 API 인증키 (이미 포함되어 있음)

## 🚀 사용 방법

### 1단계: 필요한 패키지 설치

```bash
pip install -r requirements.txt
```

또는

```bash
pip install flask flask-cors requests
```

### 2단계: 서버 실행

```bash
python server.py
```

서버가 실행되면 다음과 같이 표시됩니다:
```
🚀 서버 시작!
📍 http://localhost:5000
🔍 API: http://localhost:5000/api/search
```

### 3단계: 웹 페이지 열기

브라우저에서 `index.html` 파일을 열거나, 다음 주소로 접속:
- http://localhost:5000 (서버 상태 확인)
- 또는 `index.html` 파일을 직접 브라우저로 드래그

### 4단계: 사용하기

1. **"주변 음식점 검색하기"** 버튼 클릭
2. 네이버 API가 홍대입구역 근처 음식점을 검색
3. 300m 이내 음식점만 필터링되어 표시
4. **"랜덤 추첨하기"** 버튼으로 오늘의 점심 결정!
5. 하단 목록에서 페이지 넘기며 모든 음식점 확인 가능

## 📁 파일 구조

```
├── server.py           # Flask API 서버 (네이버 API 프록시)
├── index.html          # 웹 페이지 (프론트엔드)
├── requirements.txt    # Python 패키지 목록
└── README.md          # 이 파일
```

## 🔧 작동 원리

1. **Flask 서버**: 네이버 API를 호출하고 CORS 문제 해결
2. **거리 계산**: WGS84 → KATECH 좌표 변환 후 거리 계산
3. **필터링**: 300m 이내 음식점만 선별
4. **페이지네이션**: 5개씩 나누어 표시
5. **랜덤 추첨**: 애니메이션과 함께 무작위 선택

## 🎯 주요 기능

- ✅ 네이버 지역 검색 API 실시간 검색
- ✅ 300m 이내 정확한 거리 필터링
- ✅ 거리순 자동 정렬
- ✅ 5개씩 페이지네이션 (1,2,3... 페이지)
- ✅ 도보 시간 자동 계산 (80m = 1분)
- ✅ 애니메이션 랜덤 추첨
